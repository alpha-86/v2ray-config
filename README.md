# v2ray-config
v2ray config

1. 安装原版merlin
2. 下载对应的v2ray客户端，v2ray-linux-arm
3. 修改配置config.json，配置v2ray客户端信息
4. 复制S80v2ray到 /opt/etc/init.d
5. 启动v2ray，可以执行restart.sh
6. 执行iptable_add.sh

2. 参考routing配置 https://toutyrater.github.io/basic/routing/cndirect.html
